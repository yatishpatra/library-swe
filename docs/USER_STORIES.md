# User Stories – Library Book Management System
## Sprint 1 – Book Registration
As a librarian, I want to add a new book with its ID, title, and author so that it can be stored in the library system.
As a librarian, I want the system to prevent duplicate book IDs so that each book is uniquely identifiable.
## Sprint 2 – Borrow and Return Book
As a librarian, I want to borrow a book so that its availability status is updated correctly.
As a librarian, I want the system to prevent borrowing a book that is already borrowed to avoid conflicts.
As a librarian, I want to return a borrowed book so that it becomes available again.
## Sprint 3 – Library Report
As a librarian, I want to generate a report of all books so that I can view their ID, title, author, and current status.
